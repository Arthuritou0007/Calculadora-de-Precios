# 🖩 Calculadora de Precios

## 📖 Descripción
Esta aplicación es un software de gestión desarrollado a medida para un cliente del rubro de venta de frutas y verduras. El objetivo principal del sistema es digitalizar y optimizar las operaciones diarias del local, facilitando el cálculo de ganancias mermando las pérdidas.

## ✨ Características Principales
* Interfaz intuitiva y rápida
* No requiere conección a internet
* Es capaz de devolver 3 tipos distintos de resultados

## 🛠️ Tecnologías Utilizadas
* Python
* Flet (Framework de Interfaz Gráfica)
